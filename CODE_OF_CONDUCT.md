# Code of Conduct

## Our pledge

We want this project to be a harassment-free, welcoming space for everyone, regardless of
background or identity. We pledge to act and interact in ways that keep it open and respectful.

## Expected behaviour

- Be respectful and constructive.
- Accept and offer feedback gracefully.
- Focus on what is best for the project and its users.

## Unacceptable behaviour

- Harassment, insults, or discriminatory language.
- Sexualized advances or unwelcome attention toward other people in the project space.
- Publishing others' private information without consent.
- Spam, off-topic promotion, or deliberate disruption.

> Note on generated content: this app can produce adult/NSFW images. That is about what *you*
> generate locally, on your own machine and responsibility. This Code of Conduct governs how
> people treat *each other* in the project's spaces (issues, PRs, discussions) - it is not about
> the images you make.

## Scope

This applies in all project spaces (issues, pull requests, discussions) and when representing
the project in public spaces.

## Enforcement

Maintainers may remove, edit, or reject contributions and comments that violate this Code of
Conduct, and may block repeat or serious offenders. To report a problem, open a private
[security advisory](https://github.com/nohardcoded/personas/security/advisories/new) (it does
not have to be a security issue) or contact the maintainer via GitHub.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
